Lenguajes básicos de la Web
Tipo Definicion Logo
HTML Es un lenguaje o código estándar para crear y estructurar páginas web, su principal función
es definir el contenido y la organización de algún sitio web
CSS Este lenguaje es de diseño que se utiliza principalmente para controlar la apariencia de los
documentos, especialmente de las páginas web. Funciona como complemento del HTML
JAVASCRIPT Este a diferencia del HTML si es un lenguaje de programación que permite crear páginas web
interactivas y dinámicas; aunque es de gran importancia mencionar que esta junto con las dos
explicadas anteriormente son un gran complemento que permiten obtener un buen resultado
